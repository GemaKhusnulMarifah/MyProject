Portfolio

Analysis Of Public Sentiment On Coldplay Concerts In Indonesia Using PySpark With A Logistic Regression Algorithm

This project is trying to get a sentiment analysis about Coldplay concerts in Indonesia by using PySpark with a Logistic Regression Algorithm. The dataset is from crawling comments on YouTube that have a topic about Coldplay concerts in Indonesia.

Clusterization Of Flood-Prone Areas In DKI Jakarta Province Using Hierarchical Clustering Method

This project is making a clusterization of flood-prone areas in DKI Jakarta using the hierarchical clustering method, and the output from this project is a dashboard that shows a bar chart about fatalities by administrative city, a line chart about fatalities by sub-district, and clustering in the form of a map showing the level of flood vulnerability in each sub-district.

Comparison of the Naïve Bayes, K-Nearest Neighbor, and Support Vector Machine Methods in Breast Cancer Classification

This project uses a breast cancer dataset and tries to classify it with three different models: Naïve Bayes, K-Nearest Neighbor, and Support Vector Machine. The things that get compared in this project are the confusion matrix (true positive, false negative, false positive, and true negative) and the classification report (precision, recall, f1-score, and support).

Motor Vehicle Clusterization In East Java Using the DBSCAN Method

This project uses a dataset about the number of motorized vehicles registered by district or city and type of vehicle in East Java Province in 2018–2020 and is trying to cluster using the DBSCAN method to find a pattern or trend to see the different or the same pattern based on a total number of vehicles per year.

Comparison of MySQL Workbench Database Performance with MongoDB on Zoo Database

This project is about trying to get a comparison of database performance between MySQL Workbench and MongoDB. The things that are compared are GUI, CRUD comparison, Relational Based on Type of Database, and Query Performance Comparison.

January Temperature Prediction Model In Sidoarjo District Using Multiple Linear Regression

This project is trying to get a January temperature prediction model in Sidoarjo using multiple linear regression. Analysis using SPSS and manual calculation based on multiple linear regression.

Review On The Detection And Recognition Of Emotions From Text Using NLP

This project's result is the detection and recognition of emotion in text. This project uses data about gender equality, feminism, and sexual violence from Twitter. The data went through preprocessing and then classification using Naïve Bayes classification and recognition of emotions using NRClex. Visualization data in this project using histogram, pie chart, and wordcloud.

Gas and Smoke Detector with IoT Technology

This project is using ThingSpeak, Arduino IDE 1.8.16, and Telegram. This project tests smoke, LPG gas, and CO gas with three tests each over different distances. This equipment will test out all of the gas and notify users using a buzzer and telegram if there is anything abnormal with the neighborhood using the data gathered in real time in ThingSpeak.
